pub fn shutdown(&mut self)

Shutdown the board.

This method shut-down the board by setting the pin G46 / signal HOLD low. This has no effect when powered by USB. It only works when the board is powered using the green screw terminal (P5 on schematics).

NOTE: The pin signal is set high as start by the init() function.

NOTE: Untested
